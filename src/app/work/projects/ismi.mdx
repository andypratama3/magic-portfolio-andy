Overview
ISMI by Ismi Maulidia is a premium beauty portfolio and affiliate product catalog designed specifically for oily skin beauty enthusiasts. Built with Next.js 14 and Tailwind CSS, the platform serves as both a personal brand showcase for beauty creator Ismi Maulidia and an optimized affiliate marketing system featuring curated product recommendations, beauty insights, and social media integration. The system replaces manual product curation with automated data management while maintaining a luxury aesthetic aligned with the creator's brand.
Quantified Results
- 59+ affiliate products curated and organized across 7 beauty categories (Base, Eyes, Face, Lips, Skincare)
- Automated product data management through JSON-based database with Prisma ORM for easy updates
- SEO-optimized structure with dynamic metadata, Open Graph tags, and schema markup for improved search visibility
- Social media integration with direct links to TikTok (@itsmiy0_) and Instagram (@ismi_mlda) for cross-platform engagement
- Responsive design ensuring optimal experience across mobile, tablet, and desktop devices
- Fast performance leveraging Next.js 14 App Router, server components, and edge caching
Key Features
Product Catalog System
Dynamic product filtering by category (Base, Eyes, Face, Lips, Skincare) with detailed product cards including pricing, descriptions, affiliate links, and creator endorsements ("Ismi Pakai Ini" badges). Products sourced from major Indonesian marketplaces (Shopee, Tokopedia) with real-time price and availability data.
Beauty Content Hub
Expert tips and insights for oily skin management presented through structured content sections. Includes skincare routines, makeup application techniques, and product combination recommendations tailored to specific skin concerns.
Social Media Integration
Seamless connection to Ismi Maulidia's social profiles with prominent display of TikTok and Instagram handles. Enables visitors to follow the creator's content journey while maintaining focus on product recommendations.
Affiliate Link Management
Clean, trackable affiliate URLs for monetization through partnerships with Shopee and Tokopedia. Each product includes direct purchase links while maintaining transparent disclosure of affiliate relationships.
Modern Tech Stack
Built with Next.js 14 App React Server Components, Tailwind CSS for utility-first styling, Lucide React for icons, and Vercel Analytics for performance monitoring. JSON-based data storage with TypeScript interfaces for type safety.
Technologies Used
- Framework: Next.js 14 (App Router), React 18
- Styling: Tailwind CSS, CSS Modules
- Icons: Lucide React
- Data Storage: JSON files with TypeScript interfaces, Prisma ORM (for seeding)
- Development: TypeScript, ESLint, Prettier
- Deployment: Vercel (platform-optimized for Next.js)
- Analytics: Vercel Analytics & Speed Insights
- Package Manager: npm/pnpm/yarn
Technical Challenges
The primary challenge was balancing the luxury aesthetic requirements with functional affiliate marketing needs while maintaining optimal performance. Creating a visually appealing product catalog that didn't feel like a typical affiliate site required careful attention to:
- Visual Hierarchy: Ensuring product recommendations felt curated and personal rather than commercial
- Data Management: Implementing a maintainable system for updating 50+ products without requiring code changes
- Affiliate Transparency: Clearly disclosing affiliate relationships while preserving user trust
- Performance Optimization: Balancing rich media (product images, brand logos) with fast loading times
- Mobile Responsiveness: Ensuring the luxury experience translated effectively to smaller screens
Implementing the JSON-based product database with automated seeding scripts solved the content management challenge, allowing for easy updates without modifying core application code.
Outcome
ISMI by Ismi Maulidia successfully transformed the creator's affiliate marketing approach from scattered social media links to a cohesive, professional beauty portal. The platform serves as both a portfolio showcasing Ismi Maulidia's expertise in oily skin beauty and a optimized affiliate marketing system that provides genuine value to visitors seeking trusted product recommendations.
The system establishes credibility through transparent curation (highlighting products the creator personally uses) while providing a scalable foundation for future growth. With its modern tech stack and performance-focused architecture, the platform is positioned to evolve alongside the creator's brand while maintaining the luxurious, trustworthy experience that defines her personal brand in the beauty space.
