## Overview

Built a complete event management and ticket sales platform for Koetai Mahkota Soundline using Laravel 10. The platform handles event creation, ticket inventory management, online sales, payment processing, and sales analytics — all delivered within a tight 2-month timeline.

## Key Results

- **45% increase** in online ticket sales through UX optimization of the checkout flow
- **Zero critical bugs** in production since launch
- **Delivered ahead of schedule** despite the aggressive 2-month timeline

## Key Features

**Event Management** — Create and manage events with date scheduling, venue information, ticket categories, and pricing tiers.

**Ticket Sales** — Online ticket purchasing with real-time inventory tracking. Multiple ticket categories per event (early bird, regular, VIP, etc.) with automated capacity management.

**Payment Integration** — Integrated Midtrans Payment Gateway supporting 20+ Indonesian payment methods including Virtual Accounts, e-wallets, and credit cards. Secure transaction processing with webhook callbacks and automatic status updates.

**Admin Dashboard** — Sales monitoring, transaction history, attendee management, and Excel report export.

## Technologies Used

Laravel 10, PHP 8.1+, MySQL, Midtrans Payment Gateway, Tailwind CSS, Bootstrap
